NASA Deep Learning Computer Vision

 Project Description:

 This project investigated the use of deep learning computer vision for the task of fastener detection. Resulting from this work is a published paper available through this link (Abstract Below) and an annotated dataset of cross-recessed-screws available through this link. Below the abstract is an embedded video of an RI AI Meetup talk that I gave, which provides a summary of this project. This work was funded by the NASA Rhode Island Space Grant and supported by the URI Artificial Intelligence Laboratory.
